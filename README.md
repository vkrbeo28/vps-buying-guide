## 一、买之前，先想清楚这几件事

别急着下单。先坐下来，泡杯茶，问自己四个问题。

**第一，你买来干嘛？** 建站、跑机器人、科学上网、做跨境电商店铺、挂 PT、学 Linux……用途不同，对线路和配置的要求天差地别。建站要稳定、要大流量；跑机器人要 CPU 和内存；电商店铺要住宅 IP；学 Linux 随便一台便宜的就行。

**第二，你主要用什么网络访问？** 电信、联通、移动，三家回国线路的"高端套餐"完全不一样。电信认 CN2 GIA，联通认 CUII / AS9929，移动认 CMIN2。你买的 VPS 如果线路跟你家宽带不对付，那速度就别指望了。

**第三，能接受什么价位？** 便宜的年付几十块也有，贵的月付几百块的也有。预算决定你能在"线路等级"和"配置大小"之间怎么分配。

**第四，会不会用？** 国外 VPS 基本都是 Linux 系统，给你一个 IP 和 root 密码，剩下的自己折腾。如果你连 SSH 都不会连，那要么先学，要么老老实实买带面板的虚拟主机。

## 二、国外 VPS 的几种购买渠道，简单对比一下

市面上卖国外 VPS 的，大概分三类。

**国际大厂**：Vultr、DigitalOcean、Linode、Hetzner 这类。优点是正规、稳定、文档全、全球节点多；缺点是默认走国际线路，国内访问速度看运气，电信用户经常绕路，而且价格不算便宜，月付 4-6 美元起步。

**老牌国人商家**：搬瓦工、RackNerd 这类。搬瓦工的 CN2 GIA 套餐口碑不错，但常年缺货；RackNerd 便宜大碗，不过走的是普通 BGP，优化一般。

**专做优化线路的小众商家**：这类商家规模不大，但专门针对中国大陆做线路优化，把 CN2 GIA、CUII、CMIN2、软银、IIJ 这些高端线路拉到自己的机房里。**VMISS 就属于这一类**——2022 年在加拿大注册的公司，机房分布在香港、韩国、日本、美国、英国，主打的就是"三网优化"和"高端直连"。

如果你是新手，又特别在意国内访问速度，第三类其实是最省心的选择。不用自己研究线路怎么走，商家都给你配好了。
